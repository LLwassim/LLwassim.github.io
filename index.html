Banking Application
For Testing

I'm excited to introduce my comprehensive Banking application, a sophisticated educational tool designed to help my students learn how to emulate real-world testing scenarios within the realm of a banking application. This project leverages a robust technology stack that includes React, Express, MySQL, AWS RDBS, AWS EC2, Node.js, Passport.js, React Authentication, Material UI, and Axios,Hosted on Github Pages, Deployed on AWS EC2, Using Nginx for Reverse Proxy, With a custom domain and SSL certificate handled by OPENSSL supported by Postman for API testing.
The goal of this application is to provide a practical and immersive environment for students to gain hands-on experience in software testing, development, and deployment. It's equipped with a plethora of features and functionalities that mimic the intricacies of real banking systems. This allows students to apply their knowledge in a dynamic and interactive setting, enabling them to gain a deep understanding of the intricacies of the banking sector and how to rigorously test and maintain such systems.
One notable aspect of this project is its integration with JIRA, a widely used project management tool. It has its own JIRA project with user stories and tickets, mirroring the professional workflow in the software development industry. This empowers students to learn project management practices and agile methodologies, preparing them for the demands of the real world.
With React on the front end, students can explore responsive and user-friendly interfaces, while Express, Node.js, and MySQL on the back end provide a strong foundation for data management and server-side functionalities. Passport.js and React Authentication ensure that user data remains secure, adhering to best practices for authentication and authorization.
The user interface, designed with Material UI, offers an intuitive and visually appealing experience, making it easier for students to interact with the application. They can navigate various banking operations seamlessly, such as account management, fund transfers, transaction history, and more
Additionally, the application interfaces with external services through the Axios library, allowing students to understand how real-world systems interact with external APIs and services. Postman integration further supports students in comprehensively testing their API endpoints and ensuring their functionality meets industry standards.
AWS plays a pivotal role in this project, as AWS RDBS provides a scalable and reliable database solution, while AWS EC2 facilitates the deployment of the application on cloud servers. This mirrors the modern deployment and scaling practices widely used in the industry.
In summary, this Banking application offers a holistic educational experience, covering a wide range of skills and technologies necessary for a successful career in software development and testing. By combining a rich tech stack with real-world testing scenarios, JIRA integration, and practical project management, this application equips students with the knowledge and skills they need to thrive in the dynamic world of software development and testing.
